multiapp module

class multiapp.MultiApp

Bases: object

Framework for combining multiple streamlit applications. Usage:

def foo():

st.title(“Hello Foo”)

def bar():

st.title(“Hello Bar”)

app = MultiApp() app.add_app(“Foo”, foo) app.add_app(“Bar”, bar) app.run()

It is also possible keep each application in a separate file.

import foo import bar app = MultiApp() app.add_app(“Foo”, foo.app) app.add_app(“Bar”, bar.app) app.run()

add_app(title, func)

Adds a new application. Parameters ———- func:

the python function to render this app.

title:

title of the app. Appears in the dropdown in the sidebar.
